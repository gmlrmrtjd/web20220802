git hub 사용법(버전관리 프로그램)
: git scm설치 
-> git bas실행 
-> 폰트 d2coding 
-> $ git config --global user.name "John Doe"
-> $ git config --global user.email johndoe@example.com 
-> 확인 방법: $ git 
-> 해당 폴더 우측버튼 git bash here (제목표시줄 마지막에 해당폴더 루트 확인)
-> $ git init
-> $ git add .
-> $ git commit -m "first commit"
-> $ git status (마지막 단어 here 확인)
-> github 로그인
-> 우측 상단 '+' 버튼 클릭 후 'New repository' 클릭
-> repository name 지정 후 'create repository' 클릭
-> git remote ad ~~~ 해당 문단 복사
-> 두번쨰 열린 bash에 붙여넣기
-> $ git push -u origin master (로그인창 확인)
-> Sign in with your browser 클릭
-> Authorize 클릭
-> bash에서 마지막에 ~~/master 확인
-> git hub 새로고침 후 파일 확인
-> $ exit 종료
